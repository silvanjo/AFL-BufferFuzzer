## Überblick

Der AFL-BufferFuzezer ist ein auf AFL basierender Fuzzer, der speziell für die Aufdeckung von Buffer Overflows konzipiert ist. Er verwendet eine Instrumentierung, die zur Laufzeit Informationen über Speicherzugriffe des System under Test (SUT) sammelt. Für das Erkennen von Buffer Overflows, Buffer Overreads und Index out of Bounds Fehler werden keine weitere Sanitizer benötigt.
